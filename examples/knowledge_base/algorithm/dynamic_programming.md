# Dynamic Programming

Dynamic programming solves problems by dividing them into overlapping
subproblems. A state transition describes how to compute a larger problem from
smaller problems.

Common examples include Fibonacci numbers, shortest paths in directed acyclic
graphs, and knapsack optimization. Dynamic programming usually stores previous
states to avoid repeated computation.

中文补充：动态规划通过状态定义和状态转移来解决具有重叠子问题的任务。
